# The Game That Keeps You at the Table

WOPR runs every scenario. First strike, second strike, limited exchange, surgical strike. Each strategy is optimal. Each one fails. The sophistication is real and the impossibility is structural.

The wall does not say try harder. The wall says: the kind of approach you are using is structurally incapable of reaching the answer. The brilliance is the reason it's hard to see the blockage. People stay confused not because they're thinking poorly. Because they're thinking well - inside a space that thinking well cannot escape.
