# The Score You Actually Need

You spent the whole time chasing a ten. The perfect reconciliation. The clean win. The playbook that would fix everything. You got a five. The contaminated good. The dance with the broken person who didn't pretend to be whole.

The five was always the point. The sanctuary was never built from perfect materials. It was built from what survived the fire. The good point doesn't need to be clean. It needs to be found.
