# The Unproductive Things Were Load-Bearing

The grandmother's presence had no line item. The master's authority had no KPI. The guild's mutual obligations had no ROI. The village's web of holding had no metric at all. In a civilization that could only see what it could count, the uncountable things became invisible. Then they disappeared.

The Sabbath rest looked like inefficiency. The elder who sat with you and said nothing useful looked like wasted time. Each optimization severed a connection that had been load-bearing without anyone noticing. Each one turned out to be a load-bearing wall.
