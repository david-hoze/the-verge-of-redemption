# The Shape, Not the Feeling

Sadness is not an emotion. It is a structural deformation. The heart has a shape, and when the shape is bent, the output is sadness. A warped beam doesn't need encouragement. It doesn't need to believe in itself. It needs to be straightened.

Every self-help book treats sadness as a feeling to be managed. The 19th-century rabbi who hid an operating manual for happiness inside property law treated it as an engineering problem. Change the shape, not the story.
